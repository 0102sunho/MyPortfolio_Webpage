# README.md

Simple static webpage to introduce self

## CREATION GOAL:

Focuses on modern design rather than functionality.
Uses multiple .css and .js features to make the design smooth and modern, along with clean animations that depend on user scroll.
Has multiple sections, showcasing:
- Person's name (first and last), and a quote
- Their expertise and strength
- Their background, resume and education, etc.

Each section's text part should be placeholder, allowing a person to fill it in freely.
